Principais formatações

Negrito/Destaque

Nesta frase temos um "termo em negrito", utilizando a tag <b>, (não semantica).

Nesta frase temos um "termo em negrito", utilizando a tag <strong>,(semantica).

Itálico/Ênfase

Nesta frase temos um "termo em itálico", utilizando a tag <i>,(não semantica).

Nesta frase temos um "termo com enfase", utiizando a tag <em>,(semantica).

Texto Marcado

Nesta frase temos um texto marcado com a tag <mark>.

Nesta frase temos um texto marcado, porém utilizei a tag <style=background-color>para alterar a cor.

Texto Grande e Texto Pequeno

Nesta frase temos "esta frase"com o texto grande, utilizando a tag <big>, porém esta TAG está obsoleta.

Já nesta frase temos "esta frase"com o texto pequeno, utilizando a tag <small>.

Texto Excluido/Deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido , mas não considerado.Utilizamos a tag <del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois, utilizando a tag <ins>.

Para sublinhar um texto, utilizamos a tag <u>, porém ela está obsoleta.

Texto Sobrescrito

Para inserir coisas do tipo x20+3, utilizamos a TAG <sup>.

Texto Subescrito

Para inserir coisas do tipo H2O, utilizamos a TAG <sub>.
