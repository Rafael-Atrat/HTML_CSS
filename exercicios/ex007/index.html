Trabalhando com listas

Listas Ordenadas

Listas ordenadas podem ser criadas com a tag <ol>, e dentro desta TAG, coloco cada tarefa dentro de uma TAG <li>. Assim cadas vez que acrescentar uma tarefa, ela fica ordenada de forma correta.

  1. Acordar!
  2. Levantar
  3. Escovar os dentes!
  4. Tomar café da manhã!
  5. ir trabalhar

Listas Não Ordenadas

Listas não ordenadas podem ser criadas com a ta <ul>, também utilizando a tag <li> para cada item da lista, porem não irá numera-las.

  • item 1
  • item 2
  • item 3
